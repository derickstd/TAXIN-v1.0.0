# 📱 Taxman256 Mobile Quick Reference

## Touch Gestures

### Navigation
| Gesture | Action |
|---------|--------|
| 👆 Swipe from left edge | Open sidebar menu |
| 👉 Swipe right on sidebar | Close sidebar |
| 👆 Tap overlay | Close sidebar |
| 👇 Pull down from top | Refresh page |

### Cards & Lists
| Gesture | Action |
|---------|--------|
| 👉 Swipe right on card | Quick complete |
| 👈 Swipe left on card | Quick archive |
| 👆 Tap card | View details |
| 👆 Long press | Context menu (coming soon) |

### Tables
| Gesture | Action |
|---------|--------|
| 👈👉 Swipe horizontally | Scroll table |
| 👆👇 Swipe vertically | Scroll page |

## Tips & Tricks

### Speed Tips
- **Swipe from left** - Fastest way to open menu
- **Bottom nav** - Quick access to main sections
- **Pull to refresh** - Update data instantly
- **Swipe cards** - Quick actions without opening

### Battery Saving
- **Dark mode** - Available in settings
- **Reduce animations** - Browser setting
- **Offline mode** - Works without internet

### Productivity
- **Add to Home Screen** - Launch like native app
- **Enable notifications** - Never miss deadlines
- **Use search** - Faster than scrolling
- **Keyboard shortcuts** - Alt+key combinations

## Common Actions

### Quick Add
1. Tap **+** button in topbar
2. Or use bottom nav to navigate
3. Or swipe from left → tap "New..."

### Quick Search
1. Tap search field
2. Start typing (auto-search)
3. Tap result to open

### Quick Filter
1. Tap filter chips at top
2. Or use dropdown filters
3. Tap "Clear" to reset

## Troubleshooting

### Sidebar stuck?
- Tap overlay to close
- Or swipe right on sidebar
- Or tap hamburger icon

### Can't scroll table?
- Look for scroll indicator
- Swipe horizontally on table
- Rotate to landscape mode

### Zoom on input?
- Already prevented in iOS
- If happens, update browser

### Bottom nav missing?
- Only shows on phones (< 600px)
- Rotate to portrait mode
- Check if logged in
